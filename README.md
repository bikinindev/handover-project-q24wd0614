# Running
1. Clone the repository by open terminal or command prompt
```bash
    git clone <repository_name>
```

2. Open the folder
```bash
    cd <folder_name>
```

3. Install Dependency
```bash
    npm install
```

4. Run the development stage
```bash
    npm start
```
